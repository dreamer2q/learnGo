# alpha-demo
demos for korok engine's beta release

To install korok engine: 
```
go get korok.io/korok
```
Clone demo project:
```
git clone git@github.com:KorokEngine/beta-demo.git
```
Run demo(eg:text):

```
$ cd text
$ go run main.go
```
